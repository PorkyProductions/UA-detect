Easy to use

We built this library from the ground-up with the intention to simplify the navigator API. We started by running into a problem on one of our other websites, where the header text would show up on mobile devices, making it look messy and unreadable. We decided to solve this problem by writing some JavaScript that would scan the User Agent string, and if any attributes related to a mobile device showed up, it would hide the text. This script eventually evolved into the library that it is tody, with many more features, and more coming in the future.
